# Cú pháp

Tất cả các đoạn code Java đều phải nằm trong một class.

Java là case-sensitive, MyClass và myclass sẽ được coi là 2 class khác nhau.

Phương thức `main()` là bắt buộc trong mọi chương trình Java, Nó là entry-point để Java khởi chạy.

```java
public static void main(String[] args)
```

Luôn kết thúc câu lệnh bằng dấu `;`.
