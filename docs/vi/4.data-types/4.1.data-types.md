# Kiểu dữ liệu

### Kiểu dữ liệu nguyên thủy (primitive type)

byte, short, int, long, float, double, boolean, char.

Java lưu trực tiếp giá trị nguyên thủy vào ngăn xếp (Stack).

vd: `int i = 1;`

Nếu không gán gì cho một biến có kiểu dữ liệu nguyên thủy, Java sẽ tự gán cho nó một giá trị mặc định, thường là `0`, `'\u0000'` hoặc `false`.

### Kiểu dữ liệu không nguyên thủy (Non-primitive / Reference types)

String, Array, Class.

Còn được gọi là kiểu dữ liệu tham chiếu.

Java lưu địa chỉ trỏ đến vùng nhớ khác (Heap).

Nếu không gán gì cho một biến có kiểu dữ liệu tham chiếu, mặc định nó sẽ có giá trị `null`.

> ## __Chú ý__:

Lưu ý cực kỳ quan trọng về "Biến cục bộ" (Local Variables)

Có một cái "bẫy" mà người mới học Java rất hay gặp: Nếu bạn khai báo biến bên trong một phương thức (hàm main, hàm tính toán...), Java sẽ không tự động gán giá trị mặc định.

Quy tắc: Nếu bạn không khởi tạo giá trị cho biến cục bộ mà đã mang đi sử dụng, Java sẽ báo lỗi Compile Error ngay lập tức.

```java
public class Test {
    boolean a; // Biến thực thể (Instance variable) -> Mặc định là false

    public void check() {
        boolean b; // Biến cục bộ (Local variable) -> KHÔNG có giá trị mặc định
        
        System.out.println(a); // In ra: false (Chạy tốt)
        // System.out.println(b); // LỖI: Variable 'b' might not have been initialized
    }
}
```
