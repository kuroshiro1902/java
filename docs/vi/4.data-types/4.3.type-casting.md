# Ép kiểu

Ép kiểu (Type Casting) trong Java là việc gán giá trị của một kiểu dữ liệu này sang một kiểu dữ liệu khác. Có hai loại chính:

---

### 1. Widening Casting (Tự động)
Chuyển từ kiểu có kích thước **nhỏ hơn** sang kiểu có kích thước **lớn hơn**. Java tự động thực hiện việc này vì không làm mất dữ liệu.

* **Thứ tự:** `byte` -> `short` -> `char` -> `int` -> `long` -> `float` -> `double`
* **Ví dụ:**
    ```java
    int myInt = 9;
    double myDouble = myInt; // Tự động ép kiểu thành 9.0
    ```

### 2. Narrowing Casting (Thủ công)
Chuyển từ kiểu có kích thước **lớn hơn** sang kiểu **nhỏ hơn**. Bạn phải viết kiểu dữ liệu trong dấu ngoặc đơn `()` trước giá trị vì có nguy cơ mất dữ liệu (tràn số hoặc mất phần thập phân).

* **Thứ tự:** `double` -> `float` -> `long` -> `int` -> `char` -> `short` -> `byte`
* **Ví dụ:**
    ```java
    double myDouble = 9.78d;
    int myInt = (int) myDouble; // Thủ công ép kiểu thành 9 (mất .78)
    ```

---

### Tóm tắt nhanh
| Loại ép kiểu | Thực hiện | Kích thước | Rủi ro |
| :--- | :--- | :--- | :--- |
| **Widening** | Tự động | Nhỏ $\rightarrow$ Lớn | An toàn |
| **Narrowing** | Thủ công | Lớn $\rightarrow$ Nhỏ | Mất dữ liệu |

> **Lưu ý:** Ép kiểu chỉ áp dụng cho các kiểu dữ liệu có liên quan đến nhau (ví dụ: các kiểu số). Bạn không thể ép kiểu trực tiếp từ `boolean` sang `int`.
