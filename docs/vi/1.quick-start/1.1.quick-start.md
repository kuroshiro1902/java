# Java quick start

Chạy:
```
> java -version
```

Nếu Java đã được cài đặt, màn hình sẽ hiển thị như sau (tùy phiên bản):
```
java version "21.0.7" 2025-04-15 LTS
Java(TM) SE Runtime Environment (build 21.0.7+8-LTS-245)
Java HotSpot(TM) 64-Bit Server VM (build 21.0.7+8-LTS-245, mixed mode, sharing)
```

Tạo file `Main.java`:

``` java
public class Main {
  public static void main(String[] args) {
    System.out.println("Hello world");
  }
}
```

Chạy để biên dịch file Java thành bytecode (file `Main.class`).
```
> javac Main.java
```

Chạy lệnh để thực thi bytecode:

```
> java Main
```

Output: 
```
Hello World
```
